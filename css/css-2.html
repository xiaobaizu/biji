概述

字体超出隐藏

示例

HTML 代码:
<p>Hello, are you having a good time over there?</p>
css 代码:
p {
    width: 120px;
    white-space: nowrap;
    overflow: hidden;
    text-overflow: ellipsis;
}
如果想要限制行数
p {
    -webkit-line-clamp: 3;
    display: -webkit-box;
    -webkit-box-orient: vertical;
    overflow: hidden;
}
