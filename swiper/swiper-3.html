概述

切换Slide时执行(activeIndex发生改变)

示例

js代码
var mySwiper = new Swiper ('.swiper', {
    on:{
        slideChange: function(){
           $('p').removeClass('active');
           $('p').eq(this.realIndex).addClass('active');
        },
    },
})

如果想绑定swiper跟着切换

$('p').click(function () {
    mySwiper.slideToLoop($(this).index())
})
