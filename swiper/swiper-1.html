概述

Swiper使用方法：

第一步：加载插件,需要用到的文件有swiper-bundle.min.js和swiper-bundle.min.css文件，下载swiper-7.4.1.zip

Swiper官方网址:

https://www.swiper.com.cn/
引入你的css和js
<link rel="stylesheet" href="swiper-bundle.min.css">
<script src="swiper-bundle.min.js">/script>
点击查看css文件 点击查看js文件

示例

第二步：构建你的html结构，给Swiper定义一个大小，当然不要也行。

HTML 代码:
<div class="swiper">
    <div class="swiper-wrapper">
        <div class="swiper-slide"></div>
        <div class="swiper-slide"></div>
        <div class="swiper-slide"></div>
    </div>
    <div class="swiper-pagination"></div>
    <div class="swiper-button-prev"></div>
    <div class="swiper-button-next"></div>
</div>

如果想要禁止左右滑动，可以在你的swiper上添加一个类

<div class="swiper swiper-no-swiping"></div>
CSS 代码:
.swiper {
    width: 600px;
    height: 300px;
}
js代码
var mySwiper = new Swiper ('.swiper', {
    direction: 'vertical',
    speed: 1000,
    slidesPerView: 3,
    spaceBetween: 20,
    initialSlide: 1,
    centeredSlides: true,
    pagination: {
        el: '.swiper-pagination',
    },
    navigation: {
        nextEl: '.swiper-button-next',
        prevEl: '.swiper-button-prev',
    },
})
